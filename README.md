# ⛩️ 神社へGO

**「寒いし、人混みは嫌だ…でも初詣はしたい！」**
そんなワガママを叶える、究極のインドア初詣アプリが爆誕しました。

温かい部屋でぬくぬくしながら、お賽銭を投げて、運勢を占ってみませんか？
待ち時間0秒で、ご利益をゲットしましょう！

## 🎮 何ができるの？

1. **日本全国から選べる！**

   日本地図から、気になった神社を3箇所から選んでタップするだけ。移動時間は一瞬です。

2. **お賽銭は「投げる」スタイル**

   お賽銭をドラッグ＆ドロップして、賽銭箱へ *シュッ* と投げ入れましょう。
   この「投げ感」にこだわりました。

3. **徳を積んで運試し**

   お賽銭を投げると「お祈りポイント」が貯まります。
   5ポイント貯まると、お楽しみの「おみくじ」が引けるようになります！

4. **過去の運勢もバッチリ**

   「あれ、さっき何引いたっけ？」となっても大丈夫。
   最新10回分の結果は履歴に残るので、運勢の上がり下がりを見返して楽しめます。


## 🚀 使い方

**直感的な操作で、誰でも簡単に初詣を楽しめます。**

1. **アプリを起動＆選択**：一覧から、今すぐ参拝したい神社を選んでタップ！

2. **投げて、祈る**：選んだ神社で、お賽銭を賽銭箱へ *シュッ* と投げ入れ、お祈りします。

3. **お祈りポイントを獲得**：お賽銭を入れることで、ありがたい「お祈りポイント」をゲットできます。

4. **おみくじをひく**：貯まったお祈りポイントを消費して、ドキドキの「おみくじ」を引きます。

5. **過去のおみくじ結果の確認**：過去10回分のおみくじの結果は履歴として保存されるので、運勢を確認できます。

## ✨ アピールポイント

- **リアルな体験演出**  
ただのボタン連打ではありません。
お賽銭を投げ入れたり、おみくじを引いたりする動作に**こだわりのモーション**を導入しており、実際の初詣に近い体験を味わえます。

- **人混み不要の「快適さ」**  
人気の神社を複数収録していますが、**待ち時間はゼロ**。
長時間並ぶストレスとは無縁の、スマートな初詣を実現しました。

- **やり込み「ポイントシステム」**  
お賽銭で「お祈りポイント」を稼ぎ、それを使っておみくじを引くサイクル！
楽しみながら自然と継続して利用できる、**ゲーム的な仕組み**になっています。

- **運勢履歴の「確認機能」**  
「さっき何が出たっけ？」となっても安心。
おみくじの結果を**最大10回分まで保存**できるので、これまでの運勢の波をいつでも振り返れます。
