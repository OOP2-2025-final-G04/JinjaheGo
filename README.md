# 神社へGO

神社へGO は、自宅にいながら初詣を体験できる「仮想初詣アプリ」です。
人混みの中に出向くことなく、好きな神社を選んでお賽銭を入れたり、おみくじを引いて運勢を占うことができます。
忙しい人や混雑を避けたい人でも、気軽に初詣の雰囲気を楽しめるアプリです。

## 使い方

1. アプリを起動し、一覧から参拝したい神社を選びます

1. 選択した神社でお賽銭を入れ、お祈りを行います

1. お賽銭を入れることで「お祈りポイント」を獲得できます

1. 獲得したお祈りポイントを使って、おみくじを引くことができます

1. おみくじの結果は履歴として保存され、過去10回分の運勢を確認できます

直感的な操作で、誰でも簡単に初詣を楽しめます。

## アピールポイント

- リアルな体験演出  
お賽銭を投げ入れたり、おみくじを引いたりする動作にモーションを取り入れ、実際の初詣に近い体験ができます。

- 人混み不要の快適さ  
人気の神社を複数収録しており、長時間並ぶことなく初詣を済ませることができます。

- やり込み要素のあるポイントシステム  
お賽銭によって獲得できる「お祈りポイント」を使っておみくじを引く仕組みのため、楽しみながら継続して利用できます。

- 運勢履歴の確認機能  
おみくじの結果を最大10回分まで保存でき、これまでの運勢を振り返ることができます。
